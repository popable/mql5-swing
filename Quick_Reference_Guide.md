## 📊 HH/HL/LH/LL快速识别

### 🟢 上升趋势信号
- **Higher High (HH)**: 新高点 > 前高点
- **Higher Low (HL)**: 新低点 > 前低点
- **买入时机**: HL确认后的反弹
- **止损**: HL下方8-15点

### 🔴 下降趋势信号  
- **Lower High (LH)**: 新高点 < 前高点
- **Lower Low (LL)**: 新低点 < 前低点
- **卖出时机**: LH确认后的回落
- **止损**: LH上方8-15点

### 🟡 趋势转换信号
- **上转下**: HH失效 → LH确认 → LL突破
- **下转上**: LL失效 → HL确认 → HH突破

## 🎯 实时趋势状态应用

### 🟢 上升趋势状态
- **策略**: 专注买入机会
- **入场**: 等待HL形成
- **避免**: 追高买入
- **风控**: 在HL下方设止损

### 🔴 下降趋势状态
- **策略**: 专注卖出机会  
- **入场**: 等待LH形成
- **避免**: 抄底买入
- **风控**: 在LH上方设止损

### 🟡 区间震荡状态
- **策略**: 高抛低吸
- **买入**: 接近区间下沿
- **卖出**: 接近区间上沿
- **风控**: 快进快出，严格止损

## �️ 风险管理快速表

### 仓位计算
```
基础仓位 = (账户 × 风险%) ÷ 止损点数

剥头皮: 风险1-2%/笔, 日风险≤5%
日内: 风险1%/笔, 日风险≤3%  
波段: 风险0.5-1%/笔, 月风险≤10%
```

### 止损设置
```
技术止损: 前一个相反波段点外
时间止损: 剥头皮5分钟, 日内4小时, 波段3天
盈亏比: 剥头皮≥1:1.5, 日内≥1:2, 波段≥1:3
```

## 📞 紧急情况处理

### 重要数据发布
- **行为**: 提前30分钟关闭所有仓位
- **等待**: 数据发布后30分钟再开仓
- **观察**: 新趋势状态和波段点变化

### 系统异常
- **指标不更新**: 重启MT5或重新加载指标
- **信号冲突**: 以更大时间框架为准
- **网络问题**: 立即平仓止损

### 连续亏损
- **3笔连亏**: 停止交易1小时
- **日亏损达3%**: 停止交易当日
- **周亏损达10%**: 停止交易当周

记住：纪律胜过一切技巧！
1. 单笔风险 ≤ 1%
2. 连续亏损3单 → 暂停交易
3. 日亏损达3% → 停止交易
4. 最大同时持仓：2单
